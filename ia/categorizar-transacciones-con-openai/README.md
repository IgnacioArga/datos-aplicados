# Categorizar transacciones con OpenAI

Clasificar gastos (Alimentación, Transporte, Compras, etc.) pasándole las transacciones a un LLM. Ejemplo didáctico con datos de muestra.

**Requisitos:** `openai`, `pandas`. Necesitás una API key de OpenAI (reemplazá el placeholder `sk-...`).
**Cómo correr:** `python categorizar_transacciones_openai.py`

# Informe mensual automatizado

Caso **anonimizado**: un informe mensual que antes se armaba a mano cruzando varias planillas. El script junta las fuentes, limpia lo básico, arma el resumen por categoría y exporta el Excel final. (Datos y nombres son de ejemplo; el patrón es el real.)

**Requisitos:** `pandas`, `openpyxl`.
**Cómo correr:** poné las planillas en `fuentes/` y ejecutá `python informe_mensual.py`.

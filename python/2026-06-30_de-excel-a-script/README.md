# De Excel a script

Te llegan N planillas (una por sucursal o por mes) y las pegás a mano. Este script las une todas en segundos y deja un único archivo, agregando una columna `origen` para trazabilidad.

**Requisitos:** `pandas`, `openpyxl`.
**Cómo correr:** poné los `.xlsx` en una carpeta `entrada/` y ejecutá `python consolidar_excels.py`.

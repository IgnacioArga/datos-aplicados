# Geolocalización con Google Maps

Geocodifica direcciones de un CSV (lat/long) con la API de Google Maps y calcula distancias aproximadas a puntos de referencia. Direcciones de ejemplo, genéricas.

**Requisitos:** `pandas`, `requests`. Necesitás una API key de Google Maps (reemplazá el placeholder).
**Cómo correr:** poné tus direcciones en `direcciones.csv` (columna `address`) y ejecutá `python geolocalizacion_direcciones.py`.

# Concatenar TXT y CSV

Une todos los `.txt` (separados por `|`) y `.csv` (separados por `,`) de una carpeta en una sola tabla, normaliza headers y exporta/apendea a un CSV.

**Requisitos:** `pandas`.
**Cómo correr:** ajustá `input_folder` / `output_folder` y ejecutá `python concatenar_txt_csv.py`.

# Scraping con Selenium

Ejemplo mínimo de automatización web: abre una página en modo headless, hace click en un botón y captura un dato (p. ej. un precio). Selectores de ejemplo, a adaptar al sitio real.

**Requisitos:** `selenium`, `webdriver-manager`.
**Cómo correr:** `python selenium_ejemplo.py`

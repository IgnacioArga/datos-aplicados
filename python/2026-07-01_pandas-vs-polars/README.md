# pandas vs polars

Benchmark de un `groupby`/agregación sobre el mismo dato, a tamaños crecientes (10K → 20M filas), para ver dónde polars saca diferencia.

**Requisitos:** `numpy`, `pandas`, `polars`.
**Cómo correr:** `python benchmark_pandas_polars.py`
